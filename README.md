## Setting up locally

- Clone the rp repository

```bash
    git clone git@github.com:sanam2405/rp.git
    cd rp
```

- Install the dependencies

```bash
    npm install
```

- Run the frontend

```bash
    npm run dev
```

For a clean build

```bash
   npm run build
```
